# Русский язык · 10 класс — квест-тренажёр

**▶ Демо: https://andreycoderr.github.io/russian-quest-10/**

Интерактивный квест-тренажёр по **всей программе русского языка за 10 класс**
в формате подготовки к ЕГЭ. Сделан в стиле проекта «Учительская» (Арина Галицкая):
современный «коммерческий» дизайн с анимациями, геймификацией и тёмной/светлой темой.

## Что внутри

**12 станций · почти 100 заданий.** Каждая станция — отдельная тема:

| № | Станция | Темы |
|---|---------|------|
| 1 | Орфоэпия. Ударение | нормы постановки ударения (зад. 4) |
| 2 | Паронимы | различение близких по звучанию слов (зад. 5) |
| 3 | Лексические нормы | плеоназм, тавтология, сочетаемость (зад. 6) |
| 4 | Морфологические нормы | образование форм слова (зад. 7) |
| 5 | Синтаксические нормы | управление, согласование, обороты (зад. 8) |
| 6 | Правописание корней | чередующиеся и проверяемые гласные (зад. 9) |
| 7 | Правописание приставок | пре-/при-, з/с, ы/и (зад. 10) |
| 8 | Суффиксы. Н и НН | одна или две Н (зад. 11, 15) |
| 9 | Слитно/раздельно/дефис | НЕ и НИ, частицы, предлоги (зад. 12–14) |
| 10 | Лексика и фразеология | синонимы, антонимы, омонимы, фразеологизмы |
| 11 | Текст | стили, типы речи, средства выразительности |
| 12 | Морфемика и словообразование | состав слова, способы образования |

## Как это работает

- **Карта квеста** с анимированным прогрессом и общим рангом (от «Начало пути» до «Магистр слова»).
- **100 заданий** на 12 станциях, каждое — с разбором правила.
- **Геймификация:** звёзды за ответы, серии (🔥), всплывающее «+1», конфетти за хороший результат.
- Анимированный фон, плавные переходы между экранами, отзывчивые состояния кнопок и карточек.
- **Настраиваемый порог открытия станций:** 50% / 60% / 70% / «все звёзды» (идеальное
  прохождение). Следующая станция открывается, только если набран выбранный порог звёзд
  на предыдущей. Порог пересчитывается на лету и не сбрасывает прогресс.
- На каждый вопрос **сразу видно**, верно или нет, и **правило-объяснение**.
- За каждый правильный ответ — **звезда** ⭐. Станцию можно пройти заново для лучшего счёта.
- **Финал и грамота:** после прохождения всех 12 станций открывается грамота за весь квест —
  с именем ученика, рангом, числом звёзд и датой.
- **Экспорт результата для репетитора:** скопировать текстовый отчёт, сохранить грамоту
  картинкой (PNG) или отправить результат в Telegram.
- Прогресс **сохраняется в браузере** (localStorage) — можно вернуться позже.
- Управление с клавиатуры: **Enter / →** — следующий вопрос.

## Запуск

Чистый статический сайт без сборки. Достаточно открыть `index.html`
или опубликовать папку на GitHub Pages.

```
russian-quest-10/
├── index.html   — разметка
├── styles.css   — оформление (тёмная + светлая тема)
├── data.js      — все задания и объяснения
├── app.js       — логика квеста
└── README.md
```

## Как добавить задания

Открой `data.js` и допиши вопрос в нужную станцию:

```js
{
  q: "Текст вопроса (можно с <b>выделением</b>).",
  passage: "Необязательное предложение-пример.",
  options: ["вариант 1", "вариант 2", "вариант 3", "вариант 4"],
  answer: 0,            // индекс правильного варианта
  explain: "Объяснение правила, которое покажется после ответа."
}
```

Варианты на экране перемешиваются автоматически, поэтому `answer` всегда
указывает на правильный вариант в исходном порядке.
